# Обновленные правила работы с тир-листами Terraforming Mars

## Критические правила создания артефактов

### НИКОГДА НЕ СОКРАЩАТЬ КОНТЕНТ
- **Всегда пиши полные артефакты тир-листов**
- **Никогда не используй фразы типа "остальные карты аналогично"**
- **Каждая карта должна иметь полное описание**
- **Если не помещается в один артефакт - используй стратегию разделения**

### Стратегии для больших тир-листов

#### Вариант 1: Разделение по тирам
```
Артефакт 1: S-Tier и A-Tier карты (самые важные)
Артефакт 2: B-Tier карты 
Артефакт 3: C-Tier и D-Tier карты
```

#### Вариант 2: Постепенное построение
1. Создать артефакт с полной структурой но только S-tier
2. Использовать `update` команду для добавления A-tier
3. Продолжать до полного завершения

#### Вариант 3: Тематическое разделение
```
Артефакт 1: Экономические и производственные карты
Артефакт 2: Терраформирующие и событийные карты
Артефакт 3: Биологические и научные карты
```

### Обязательная структура описания карт

```markdown
### Название карты (стоимость)
**Метки:** Список меток
**Требование:** Требования (если есть)
**Эффект:** Полное описание эффекта

**Почему в X-tier:** Подробное обоснование с цифрами и примерами

**Лучшее применение:** Конкретные сценарии использования

**Синергии:** Список синергичных карт с объяснениями
```

## Принципы оценки карт

### Математические основы
- **S-tier**: Карты с ROI 200%+ или уникальными game-changing эффектами
- **A-tier**: Карты с ROI 150-200% или очень сильными специализированными эффектами  
- **B-tier**: Карты с ROI 100-150% или хорошими ситуативными эффектами
- **C-tier**: Карты с ROI 50-100% или очень ситуативными эффектами
- **D-tier**: Карты с ROI <50% или почти всегда неэффективными

### Факторы корректировки рейтинга
- **Время игры**: Производство ценнее в длинных играх, ПО в коротких
- **Количество игроков**: Атакующие карты сильнее в играх на 2 игроков
- **Дополнения**: Colonies кардинально меняет ценность энергии
- **Синергии**: Карты в синергичных стратегиях поднимаются на 0.5-1 тир

## Использование мнений сообщества

### Источники приоритетов
1. **Reddit r/TerraformingMarsGame** (основной источник)
2. **BGG форумы** (дополнительный источник)
3. **Мнения дизайнеров** (высший приоритет для спорных случаев)

### Классификация мнений
- **"One of the best"** → S-tier кандидат
- **"Very strong"** → A-tier
- **"Pretty good"** → B-tier  
- **"Situational"** → C-tier
- **"Trap card", "Never play"** → D-tier

### Weighted opinions
- **Дизайнеры карт**: 3x вес
- **Регулярные эксперты**: 2x вес
- **Новые аккаунты**: 0.5x вес

## Технические требования

### Команды артефактов
- **create**: Только для новых тир-листов
- **update**: Для изменений <20 строк в <5 местах
- **rewrite**: Для структурных изменений

### Цитирование источников
```markdown
**Почему в X-tier:** "Цитата из сообщества" согласно анализу Reddit. 
Математически карта дает X М€ ценности за Y М€ стоимости.
```

### Обязательные разделы
- Введение с объяснением критериев
- Каждый тир с объяснением характеристик
- Полные описания всех карт без исключений
- Заключение с общими рекомендациями

## Специфика разных типов тир-листов

### Корпорации
- Учитывать стартовые ресурсы и долгосрочные эффекты
- Анализировать гибкость vs специализацию
- Оценивать зависимость от удачи

### Прелюдии  
- Рассчитывать математическую ценность эффектов
- Учитывать длину игры (Prelude = короче)
- Анализировать синергии с корпорациями

### Проекты
- Основной фокус на экономическую эффективность
- Учитывать timing и требования
- Анализировать скейлинг с разными стратегиями

## Контроль качества

### Красные флаги
- Все карты одного типа в одном тире
- Отсутствие D-tier карт
- Слишком много S-tier карт  
- Противоречия в описаниях

### Финальная проверка
1. Каждая карта описана полностью
2. Обоснования логичны и последовательны  
3. Синергии указаны корректно
4. Нет дубликатов карт
5. Математика проверена

## Обновление тир-листов

### Периодичность
- После изменений в мета-игре
- При появлении новых дополнений
- При обнаружении ошибок в оценках

### Процесс обновления
1. Анализ новой информации
2. Перемещение карт между тирами если нужно
3. Обновление описаний
4. Полная перепроверка согласованности

Эти правила обеспечивают создание качественных, полных и полезных тир-листов без сокращений и пропусков.
